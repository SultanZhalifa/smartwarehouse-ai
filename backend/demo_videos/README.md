# Demo Videos

Place demo video files here for the Live Monitor to use:

| File | Used By | What |
|------|---------|------|
| `Cat.mp4` | Zone B (Storage Area) | Cat detection demo |
| `Gecko.mp4` | Zone C (Loading Dock) | Gecko detection demo |
| `Snake.mp4` | Zone D (Entrance Gate) | Snake detection demo |

## Where to Get Demo Videos

Free stock video sources:
- [Pexels](https://www.pexels.com/search/videos/) — search "snake on ground", "cat", "gecko"
- [Pixabay](https://pixabay.com/videos/) — same animals

## Recommendations

- **Format:** MP4
- **Duration:** 10-60 seconds (loops automatically)
- **Resolution:** 720p or 1080p
- **Content:** Animal clearly visible, decent lighting, indoor floor preferred for warehouse realism

The videos are NOT committed to git (too large for the repo). You provide your own.

## Webcam Alternative

Zone A defaults to webcam (`source: "0"`). If you don't have demo videos, you can still test detection by toggling Zone A on and showing your laptop camera at a printed image of any tracked class.
